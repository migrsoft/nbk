NBK Windows 版使用说明

按键操作说明

h			载入主页
alt + 左	后退
alt + 右	前进

1	适屏显示
2	100%视图
3	150%视图

z	改变工作模式（直连，云全，云简）
x	改变云工作模式（全，简）
n	夜间模式切换
r	刷新页面

当浏览模式为：云全
上w、下s、左a、右d	移动视图

当浏览模式为：本地、云简、云Wap2
上、下		焦点切换
左、右		上下翻页
enter		选中焦点


配置文件说明

配置文件为 nbk.ini，需与nbk.exe为相同目录（配置项详见nbk.ini）


wap白名单
将白名单数据存于数据目录 nbk_wl.dat 文件即可。
